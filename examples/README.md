# PhoneDo Example Scripts

The scripts in this folder are examples a user can try. They are grouped into folders by
topic (Console, Device & Hardware, File System, Network & WiFi, and so on).

To run one, open a script, copy its contents into the PhoneDo shell, and run it. Each file
has a short header describing what it does.

The `Diagnostics` folder holds scripts that check specific behaviors of the runtime.
